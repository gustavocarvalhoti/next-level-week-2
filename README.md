## Prerequisites
- [Node LTS](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)
- [Expo](https://docs.expo.io/)

## Cloning project
```bash
  # Clone repository
  $ git clone https://github.com/gustavocarvalhoti/next-level-week-2.git
  # Enter folder
  $ cd next-level-week-2
```

## Install back-end
```bash
  # Enter folder
  $ cd server
  # Install dependences
  $ yarn
  #Execute
  $ yarn start

```

## Install front-end
```bash
  # Enter folder
  $ cd web
  # Install dependences
  $ yarn
  #Execute
  $ yarn start
```

## Install mobile
```bash
  # Enter folder
  $ cd mobile
  # Install dependences
  $ yarn
  #Execute
  $ yarn start
```
